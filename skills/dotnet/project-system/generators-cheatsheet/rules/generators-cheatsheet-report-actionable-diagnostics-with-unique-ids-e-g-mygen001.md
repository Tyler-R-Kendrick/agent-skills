## Report actionable diagnostics with unique IDs (e.g., `MYGEN001`) and clear message formats

Report actionable diagnostics with unique IDs (e.g., `MYGEN001`) and clear message formats: that include the offending symbol name so developers can locate and fix the issue without reading generator source code.
