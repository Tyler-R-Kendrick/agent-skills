Best practices and rules for Source Generators Cheatsheet.

| 3 | Use `ForAttributeWithMetadataName` instead of `CreateSyntaxProvider` when filtering by attribute | MEDIUM | [`generators-cheatsheet-use-forattributewithmetadataname-instead-of.md`](generators-cheatsheet-use-forattributewithmetadataname-instead-of.md) |
| 4 | Return value types or records with value equality from transform steps | MEDIUM | [`generators-cheatsheet-return-value-types-or-records-with-value-equality-from.md`](generators-cheatsheet-return-value-types-or-records-with-value-equality-from.md) |
| 5 | Emit the marker attribute via `RegisterPostInitializationOutput` | CRITICAL | [`generators-cheatsheet-emit-the-marker-attribute-via.md`](generators-cheatsheet-emit-the-marker-attribute-via.md) |
