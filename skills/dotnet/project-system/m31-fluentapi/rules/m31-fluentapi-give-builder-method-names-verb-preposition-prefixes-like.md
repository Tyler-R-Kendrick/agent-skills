## Give builder method names verb-preposition prefixes like `WithName`, `ForAmount`, `InDepartment`, `UsingMethod`

Give builder method names verb-preposition prefixes like `WithName`, `ForAmount`, `InDepartment`, `UsingMethod`: to create readable fluent sentences; avoid generic names like `SetName` that do not flow naturally.
