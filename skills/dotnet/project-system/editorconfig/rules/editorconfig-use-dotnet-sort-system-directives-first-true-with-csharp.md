## Use `dotnet_sort_system_directives_first = true` with `csharp_using_directive_placement = outside_namespace:warning`

Use `dotnet_sort_system_directives_first = true` with `csharp_using_directive_placement = outside_namespace:warning`: for a consistent, predictable ordering of `using` directives across all files.
