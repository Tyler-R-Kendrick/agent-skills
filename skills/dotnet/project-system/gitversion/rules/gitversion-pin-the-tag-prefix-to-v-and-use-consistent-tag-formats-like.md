## Pin the `tag-prefix` to `v` and use consistent tag formats like `v1.2.3`

Pin the `tag-prefix` to `v` and use consistent tag formats like `v1.2.3`: so that GitVersion can reliably parse version tags and distinguish them from other tags in the repository.
