## Choose Mainline mode for trunk-based development and ContinuousDelivery mode for GitFlow

Choose Mainline mode for trunk-based development and ContinuousDelivery mode for GitFlow: to align version semantics with your branching strategy; mismatched modes produce confusing pre-release labels.
