## Use the `generated` symbol type with `"generator": "now"` for copyright years and `"generator": "guid"` for correlation IDs

Use the `generated` symbol type with `"generator": "now"` for copyright years and `"generator": "guid"` for correlation IDs: to inject dynamic values that are computed at template instantiation time.
