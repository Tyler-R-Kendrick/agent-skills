# SideWaffle

## Overview
SideWaffle is a collection of item and project templates for Visual Studio and Visual Studio for Mac.

## Usage
- Install via Visual Studio Extension Manager
- Access templates through Add New Item
- Custom templates for common patterns
- Project scaffolding

## Templates Include
- Web API controllers
- AngularJS components
- Nancy modules
- SignalR hubs
- Class libraries

## Best Practices
- Use templates for consistency
- Customize templates for team standards
- Create organization-specific templates
- Share templates across teams
