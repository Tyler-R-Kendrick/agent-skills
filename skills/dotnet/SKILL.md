# .NET Skills

A comprehensive collection of guidance for .NET development, covering frameworks, libraries, patterns, and best practices across the .NET ecosystem.

- **Data**: Entity Framework Core, Dapper, Redis
- **Messaging**: NServiceBus, MassTransit, Rebus, Wolverine, Brighter
- **Cloud**: Azure Functions, Azure AI Inference, Aspire
- **UI**: MAUI, Avalonia, Uno Platform, Blazorise
- **Testing**: Testcontainers, AutoFixture, Moq, Pact, Reqnroll
- **Observability**: Serilog, NLog, OTLP Logging, OpenFeature
- **Patterns**: CQRS, Event-Driven, Functional Programming, Parse Don't Validate
- **Libraries**: Polly, FluentValidation, MediatR, AutoMapper, Mapperly
- **Languages**: C# (.NET Cheatsheet, Generators), F#, Language.Ext
