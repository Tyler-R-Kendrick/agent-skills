## Set `Microsoft.AspNetCore` and `Microsoft.EntityFrameworkCore`

Set `Microsoft.AspNetCore` and `Microsoft.EntityFrameworkCore`: to `Warning` or higher in production to reduce noise from framework-internal logs.
