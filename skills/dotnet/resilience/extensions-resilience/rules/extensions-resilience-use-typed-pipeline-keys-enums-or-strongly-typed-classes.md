## Use typed pipeline keys (enums or strongly-typed classes) with `AddResiliencePipeline<TKey>`

Use typed pipeline keys (enums or strongly-typed classes) with `AddResiliencePipeline<TKey>`: instead of magic strings to prevent typos and enable compile-time verification of pipeline names.
