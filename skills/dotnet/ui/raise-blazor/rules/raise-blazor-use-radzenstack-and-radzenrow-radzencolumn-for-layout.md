## Use `RadzenStack` and `RadzenRow`/`RadzenColumn` for layout

Use `RadzenStack` and `RadzenRow`/`RadzenColumn` for layout: instead of raw HTML `div` elements with CSS classes, because Radzen's layout components respect the active theme's spacing tokens, breakpoints, and gap sizes, producing consistent density across theme switches.
