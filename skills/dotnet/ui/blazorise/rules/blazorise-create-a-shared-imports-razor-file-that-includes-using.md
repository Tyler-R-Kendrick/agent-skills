## Create a shared `_Imports.razor` file that includes `@using Blazorise` and `@using Blazorise.DataGrid`

Create a shared `_Imports.razor` file that includes `@using Blazorise` and `@using Blazorise.DataGrid`: at the project level rather than repeating these directives in every component file, reducing noise and ensuring consistent namespace availability.
