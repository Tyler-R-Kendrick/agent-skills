## Implement `Shell` navigation with `QueryProperty` or `IQueryAttributable` for page parameters

Implement `Shell` navigation with `QueryProperty` or `IQueryAttributable` for page parameters: instead of passing data through constructors or static properties; `Shell.GoToAsync` with a dictionary parameter bag supports deep linking, back-stack management, and URI-based routing.
