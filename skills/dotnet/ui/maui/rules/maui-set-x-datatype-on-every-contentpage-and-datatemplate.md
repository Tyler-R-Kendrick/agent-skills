## Set `x:DataType` on every `ContentPage` and `DataTemplate`

Set `x:DataType` on every `ContentPage` and `DataTemplate`: to enable compiled bindings, which provide compile-time type checking and avoid the `BindingBase.EnableCollectionSynchronization` warnings that appear when bindings fall back to reflection.
