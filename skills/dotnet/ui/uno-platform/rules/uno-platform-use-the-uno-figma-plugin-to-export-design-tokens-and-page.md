## Use the Uno Figma plugin to export design tokens and page layouts directly into XAML

Use the Uno Figma plugin to export design tokens and page layouts directly into XAML: rather than manually translating mockups; this eliminates pixel-level discrepancies and generates styles, colors, and typography resources that match the design system exactly.
