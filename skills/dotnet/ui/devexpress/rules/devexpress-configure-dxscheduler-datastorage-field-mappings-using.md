## Configure `DxScheduler.DataStorage` field mappings using `nameof()` expressions

Configure `DxScheduler.DataStorage` field mappings using `nameof()` expressions: instead of hardcoded strings, so that property renames on the appointment or resource model produce compile-time errors rather than silent runtime failures.
