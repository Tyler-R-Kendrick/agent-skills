## Use the Telerik ThemeBuilder (`themebuilder.telerik.com`) to generate custom SCSS overrides

Use the Telerik ThemeBuilder (`themebuilder.telerik.com`) to generate custom SCSS overrides: and include the compiled CSS as a static asset, rather than overriding Telerik CSS classes directly; direct overrides break on version upgrades when internal class names change.
