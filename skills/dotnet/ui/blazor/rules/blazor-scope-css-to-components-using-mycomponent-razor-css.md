## Scope CSS to components using `MyComponent.razor.css` isolation files

Scope CSS to components using `MyComponent.razor.css` isolation files: and reference the generated `{Assembly}.styles.css` bundle in the layout rather than writing global CSS rules that collide across component libraries.
